# 🎬 Netflix Movies & TV Shows Analysis Dashboard | Tableau

## 📌 Overview

This project provides a comprehensive analysis of Netflix’s global content library using interactive Tableau visualizations. It helps in understanding content distribution, genre trends, ratings, and growth patterns across different countries.

## 🚀 Key Features

* 🌍 Country-wise content distribution
* 🎭 Genre analysis and popularity trends
* ⭐ Ratings breakdown (TV-MA, PG, etc.)
* 📈 Growth of content over time
* 🎬 Movies vs TV Shows comparison

---

## 💡 Key Insights

* USA dominates content production
* Documentaries & Comedy are top genres
* TV-MA is the most common rating
* Rapid growth in content after 2015
